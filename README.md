# Hostel Room Allocation System

A web-based Hostel Room Allocation System developed using Flask and MongoDB.  
This system helps manage hostel room assignments, student records, and complaints efficiently.

## 🚀 Features

- Student Registration & Login
- Admin Login
- Room Allocation Management
- Complaint Submission System
- PDF Report Generation
- MongoDB Database Integration

## 🛠 Technologies Used

- Python
- Flask
- MongoDB
- HTML
- CSS
- Bootstrap
- PyMongo
- ReportLab

## ⚙ Installation Steps

1. Clone the repository:
   git clone (https://github.com/ishwari-sapkal/Hostel-Room-Allocation-System.git)

2. Navigate to project folder:
   cd Hostel-Room-Allocation-System

3. Install dependencies:
   pip install -r requirements.txt

4. Run the application:
   python app.py

5. Open in browser:
   http://127.0.0.1:5000/
